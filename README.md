🏆 Кімната Рекордів (Workout Tracker)
Персональний веб-додаток для відстеження прогресу в калістеніці, силових тренуваннях та бігу. Створений для ведення детальної статистики, розрахунку відносної сили та візуалізації змін тіла.

🚀 Основні можливості
Інтелектуальні рекорди: Система автоматично визначає найкращі результати за допомогою "Силового Індексу" (враховує вагу тіла, додаткову вагу та повторення).

Розрахунок 1ПМ: Автоматичне прогнозування теоретичного максимуму на 1 повторення за формулою Еплі.

Спортивний статус: Оцінка рівня підготовки на основі коефіцієнта відносної сили (від "Рекрута" до "Еліти").

Аналітика тіла: Трекер ваги, розрахунок ІМТ та інтерактивна "Карта пропорцій" із замірами м'язів.

Візуальний прогрес: Власна галерея фото та динамічні графіки результатів за допомогою Chart.js.

Безпека та Cloud: Синхронізація з Firebase, пакетні транзакції для надійності даних та режим адміністратора для захисту від сторонніх змін.

🛠 Технології
Frontend: HTML5, CSS3 (Modern Glassmorphism), Vanilla JavaScript.

Backend/DB: Firebase Firestore, Firebase Auth.

Libraries: Chart.js, Canvas-confetti.

⚙️ Як запустити локально
Клонуйте репозиторій.

Створіть у корені файл firebase-config.js за зразком:

JavaScript
export const firebaseConfig = {
apiKey: "ВАШ_КЛЮЧ",
authDomain: "PROJECT.firebaseapp.com",
projectId: "PROJECT_ID",
storageBucket: "PROJECT.appspot.com",
messagingSenderId: "ID",
appId: "APP_ID"
};

Відкрийте index.html за допомогою VS Code Live Server (порт 5500 за замовчуванням).

Що це дає на GitHub?
Приватність: Твої ключі в безпеці, бо вони у firebase-config.js, який прописаний у .gitignore.

Порядок: Люди бачать опис, технології та скріншоти (якщо додаси), але не мають доступу до твоєї бази.
